【制作使用エンジン】Unity

【制作期間】３ヶ月

【バージョン管理】SourceTree

【ゲームジャンル】２Ｄ・３Ｄスクロールシューティングゲーム

【ゲーム概要】
・このゲームは向かってくる隕石や敵などから回避し、射撃で破壊してスコアを稼ぐスクロールシューティングゲームです。

・プレイヤーはライフを５個所持しており、敵や敵の弾、隕石などにぶつかるとプレイヤーのライフが１個づつ減り、全てのライフが無くなると「ゲームオーバー」になります。

・プレイヤーのライフは、敵や隕石を破壊するとたまに出現するエネルギータンクを取る事で、ライフが１個回復します。

・ゲームはステージ３まであり、各ステージには「フェーズ１」「フェーズ２」「フェーズ３」が用意されております。

・各フェーズ毎には、エリアボスが存在し、エリアボスを倒すことによって次のフェーズへと移行し、「フェーズ３」のボスを撃破する事でステージクリアとなります。

・「フェーズ１」は縦スクロール、「フェーズ２」は横スクロール、「フェーズ３」は奥行のスクロールシューティングゲームとなっています。

【操作説明】
・タイトル画面やポーズ画面などの項目に、マウスのカーソルを合わせて「右クリック」で決定を行います。

・「ゲーム画面」
	・上移動ボタン：Ｗキー又は、↑キー（上矢印）
	・左移動ボタン：Ａキー又は、←キー（左矢印）
	・下移動ボタン：Ｓキー又は、↓キー（下矢印）
	・右移動ボタン：Ｄキー又は、→キー（右矢印）
	・射撃ボタン　：ＳＰＡＣＥキー
	・ポーズボタン：ＥＳＣキー

【アピールポイント・苦労した点】
・実務を想定したチーム制作（プランナー、デザイナー）と考えて制作しました。

・シューティングゲームで、単調で飽きないように２D（縦と横スクロール）と３D（奥行の立体スクロール）で遊べるゲームを制作しました。

・敵の動きが単調にならないように様々なパターンを用意しました。

・一つ一つ障害物や敵の配置を設定する作業に時間が掛かりました。さらに、障害物や敵の配置で、ゲームバランスを調整するのにも苦労しました。
